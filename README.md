# wicked-sales-js
A full stack Node.js and React shopping cart app.

### Technologies Used
- Javascript (ES5, ES6)
- React.js
- Node.js
- Express
- PostgreSQL
- npm
- Webpack
- Babel
- CSS3
- HTML5
- Bootstrap 4

### Features
- User can view items from a product list
- User can view product details from a product list
